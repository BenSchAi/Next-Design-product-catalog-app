# Next Design - Smart Product Catalog 🔎

אפליקציית קטלוג חכם לסוכנים המחוברת ל-Google Drive.

## מבנה הפרויקט:
* `app.py`: הקוד הראשי של האפליקציה (Streamlit).
* `constants.py`: מכיל את מפתח הגישה המוצפן (Base64) ל-Google Cloud.
* `requirements.txt`: רשימת הספריות הנדרשות להרצת האפליקציה.

## הוראות תחזוקה:
במידה ומחליפים חשבון שירות (Service Account), יש לעדכן את מחרוזת ה-Base64 בקובץ `constants.py`.
